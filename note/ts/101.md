# 重新学基础

## 函数签名

### 使用 `type` 声明

```ts
type Add = (x: number, y: number) => number;
```

### 使用 `interface` 声明

```ts
interface Add {
  (x: number, y: number): number;
}
```

### 配合泛型定义函数签名

定义一个函数签名，使得返回值的类型和参数类型一致

```ts
interface FuncType<T extends string | number> {
  (x: T): T;
}
```

或者使用 type

```ts
type FuncType<T extends string | number> = (x: T) => T;
```

### 给 Hook 或组件定义函数签名

```ts
interface UseDiffProps {
  editor: monaco.editor.IStandaloneCodeEditor;
  range: monaco.Range;
  modifiedText: string;
}
```

```ts
interface SelectorProps {
  filterTypes: {
    excludedFolderTypes: FolderTypes[];
    customExtension?: string[];
  };
  onSelect: () => void;
}
```
